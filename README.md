# Echo

A fun React Native game built with **Expo** where players control a collector to catch falling shapes. Match the collectorâ€™s shape with the falling shape to score points. Mismatching ends the game!

## Demo

You can add a demo video of gameplay (upload to YouTube or share a Google Drive link).

## Features

- **Tilt controls**: Move the collector by tilting your device (using accelerometer).  
- **Dynamic shapes**: Falling cubes and spheres with distinct colors.  
- **Shape switching**: Collector changes shape periodically to increase difficulty.  
- **Score system**: Gain points for correct catches and lose points for missed shapes.  
- **Game over screen**: Displays final score and option to restart.
